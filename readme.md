#### Локальный запуск для разработки и тестирования:
`yarn run webpack-dev-server`

#### Сборка для прода:
`yarn run webpack`

-------------------------------

#### Для получения ошибок валидации с сервера:
* Введите "1111-1111-1111-1111" в поле "Номер карты";
* Введите "11/11" в поле "Срок действия".
